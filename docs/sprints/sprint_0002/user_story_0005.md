# User Story 0005: Incident Reporting & Type Selection

## As a citizen, I want to categorize my emergency quickly so that the right responders are dispatched.

## Acceptance Criteria
- [x] Screen 2 shows Police, Ambulance, and Fire selection cards.
- [x] Tapping a card highlights it and auto-advances to the form.
- [x] Screen 3 (Report Form) shows auto-detected location.
- [x] Form includes optional "Add Photo" and "Description" fields.
- [x] "SUBMIT NOW" button is clearly visible and red.
