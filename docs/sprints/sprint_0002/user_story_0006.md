# User Story 0006: Live Tracking & Resolution UI

## As a citizen, I want to see the responder's progress on a map so that I know exactly when help will arrive.

## Acceptance Criteria
- [x] Tracking view shows real-time progress bar and ETA.
- [x] Map markers for User, Responder, and Destination.
- [x] Responder marker simulates movement towards the user.
- [x] Auto-advance to "Arrived" and "Resolved" states for the prototype.
- [x] Resolution Screen shows a detailed summary (Type, Time, Duration).
- [x] "Back to Home" resets the flow to the SOS screen.
