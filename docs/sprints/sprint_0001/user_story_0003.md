# User Story 0003: Dispatch & Navigation UI

## As a responder, I want to see detailed incident info and navigation shortcuts so that I can respond effectively once I accept a call.

## Acceptance Criteria
- [x] "En Route" view showing detailed location, landmark, and description.
- [x] Navigation shortcut (calls external Map app).
- [x] Communication buttons: [CALL] and [CHAT].
- [x] "On Scene" view showing status update actions and backup requests.
- [x] Data is realistically presented (Manila/Tondo context).
