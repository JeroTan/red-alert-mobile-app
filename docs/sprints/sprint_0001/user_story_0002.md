# User Story 0002: Incident Interaction UI

## As a responder, I want to see incoming incident cards so that I can accept or decline dispatches.

## Acceptance Criteria
- [x] "Nearby Emergency" card appearing below the map.
- [x] Card includes: Type (Ambulance/Fire/Police), Location, Time, and Reporter Name.
- [x] Action buttons: [ACCEPT] (Toyota Red) and [DECLINE] (Secondary).
- [x] Card styling includes a color-coded left border based on incident type.
- [x] Smooth transitions between "Available" state and "Incoming Dispatch" state.
